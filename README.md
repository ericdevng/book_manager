# Book Manager 

Aplicación web para la gestión de libros y usuarios, desarrollada en PHP como proyecto personal para practicar programación backend, manejo de sesiones y base de datos.

## Características

- Registro e inicio de sesión de usuarios
- Autenticación con sesiones
- Gestión de usuarios
- Gestión de libros
- Eliminación de cuenta de usuario
- Manejo de archivos (PDFs asociados a los libros)

## Tecnologías utilizadas

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- Composer
- Node.js (para herramientas de desarrollo)
